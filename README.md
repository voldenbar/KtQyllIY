# 项目简介

基于PyQt的深度学习面部口罩检测识别系统，主要用于识别公共场所未佩戴口罩的人员。系统功能模块包括实时视频流处理、面部检测、口罩识别及预警。技术特点方面，采用深度学习算法，通过PyQt进行界面设计，实现高效的GPU加速计算，保证实时性和准确性。此系统不涉及用户数据存储和隐私泄露，专注于提升公共安全卫生水平。


## 核心功能
- 实时视频流处理
- 面部检测
- 口罩识别
- 预警功能

## 适用场景
- 公共场所未佩戴口罩人员识别
- 提升公共安全卫生水平

## 技术特点
- 采用深度学习算法
- 使用PyQt进行界面设计
- 实现GPU加速计算
- 保证系统实时性和准确性
- 不涉及用户数据存储和隐私泄露
